This package is used to retrieve and parse a target device's OXPd Discovery Tree.

The OXPd service libraries provided in this SDK ({@link com.hp.oxpdlib.deviceinfo.OXPdDeviceInfo}, {@link com.hp.oxpdlib.scan.OXPdScan}, {@link com.hp.oxpdlib.print.OXPdPrint}, {@link com.hp.oxplib.uiconfiguration.OXPdUIConfiguration) make use of a device feature called the 'OXPd Discovery Tree' to determine whether the target device supports the required OXPd service and version, as well as the URL path to the required service. Those service libraries must be initialized with the OXPd Discovery Tree before they can be used.
