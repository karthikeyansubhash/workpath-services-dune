Automatic API list generator v0.1

Automatically generates API 4 (or any other provided) APIs list.
To launch:
1. Import into IntelliJ Idea (build should be successful, if not - fix network issues, e.g. invalid certificate, IDE settings etc.)
2. Create launch configuration with SinceApiParser as launch class
3. Select 'Working directory' for that configuration to this module folder exactly
4. Launch and observe result under Run tab
